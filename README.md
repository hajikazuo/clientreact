# Projeto de cadastro de alunos utilizando React e API Web Asp Net core

Você pode encontrar o repositório da API em: https://github.com/hajikazuo/AlunosApi

### Stack utilizada:
- .Net 8
- C# 11
- API
- Swagger
- OpenApi
- JWT
- MSSQL
- EntityFramework
- Identity
- Axios
- React
